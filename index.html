Medicaid SDK

Overview

Welcome to the Software Development Kit (SDK) for the medicaid.gov API. This notebook walks through installing the SDK and demonstrates its qualitative and analytical capabilities. The SDK was designed to be a lightweight, portable, and easy-to-use Javascript library to facilitate analysis of the vast amount of data in the Medicaid datastore. The NADAC (National Average Drug Acquisition Cost) dataset will be used as a reference throughout the remainder of the notebook, as it is among the most relevant and useful datasets in the Medicaid API.

Installation

MedicaidSDK = await import ("https://kunaalagarwal.github.io/medicaid/sdk.js");

Include the SDK in your project by installing it via ES6 imports.

Try it Yourself

Examples

Explore the examples directory to find sample applications and code snippets demonstrating how to integrate the Medicaid SDK into different scenarios. These examples serve as a starting point for your development process and showcase various use cases.

Here's an example usage:


// Get the filtered data from distribution
sdk.getDatastoreQueryDistributionId('11196f15-1a77-5b80-97f3-c46c0ce19894', 10000);

// Get all the data from a distribution or dataset
sdk.getAllDataFromDistribution('11196f15-1a77-5b80-97f3-c46c0ce19894')

// Execute a SQL query using the Medicaid SDK
let sql = '[SELECT * FROM 11196f15-1a77-5b80-97f3-c46c0ce19894][WHERE state = "Iowa"]';
sdk.getDatastoreQuerySql(sql);
